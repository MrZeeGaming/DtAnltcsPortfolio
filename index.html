Midterm Task 1.
Data Cleaning and Preparation using Excel

  This task involves cleaning and preparing a raw dataset in Microsoft Excel to improve its quality and usability. The process starts by examining the data to identify common issues such as duplicates, missing values, inconsistent formatting, and incorrect entries. Each problem is then addressed using Excel tools like Conditional Formatting, Text to Columns, TRIM, and Find and Replace. The data is organized by separating combined fields, removing extra spaces, standardizing text case, and correcting invalid values. Spelling and formatting are also reviewed to ensure consistency. The final result is a clean and structured dataset that supports accurate analysis and better decision-making.

Midterm Task 2.
Data Analysis using Pivot Table

  In this lab, you will work with a bicycle sales dataset to examine customer purchasing patterns across different countries, age groups, and genders. The first part focuses on creating a pivot table to summarize order quantities and organize data for readability. You will learn to resize columns, filter fields, sort data, and handle blank cells for clarity.

  The second part emphasizes data visualization. You will transform the pivot table into a stacked column chart to highlight trends and comparisons. This allows analysis of which demographic groups and countries generate the most sales, supporting strategic business decisions. Through this exercise, you will practice rearranging fields, refining tables, and interpreting chart data to derive meaningful conclusions about sales patterns and market opportunities.

Midterm Task 3.
Data Prep and Cleaning and Modeling w Power Query

  In this lab, you will clean and prepare the student dataset by removing null entries, assigning proper data types, and splitting or renaming columns for clarity. Power Query will then be used to separate the dataset into multiple entities, including Students, Courses, and Programs. You will create primary keys, establish foreign key relationships, and ensure the tables are free of duplicates. The final output consists of three normalized, clean, and well-structured tables that reduce redundancy and enable efficient data management and analysis.
